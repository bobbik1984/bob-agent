Features

10 Core Features

Discarding bloated web wrappers and heavy frameworks, we craft a true productivity foundation with minimalist, practical features that seamlessly integrate into your daily workflow.

Instant Inspiration Capture

Use a hotkey to instantly summon a lightweight input box at the edge of your screen. Share files or screenshots with bob anytime, as easily as messaging a friend.

Quick Summon Drag & Drop
  1. Press Ctrl+Shift+B to instantly summon the edge input bubble.
  2. Type your inspiration directly, or drag and drop files and screenshots.
  3. Press Enter, and bob automatically analyzes or smartly stores it in the background.

Smart Multi-Model Routing

bob automatically assigns models based on task complexity. Complex creations go to expert models, while background tasks and memory organization run on low-cost models, saving you money.

Smart Delegation Cost Effective
  1. Enter any third-party API keys in the settings panel.
  2. For complex text polishing or analysis, bob auto-routes to expert models.
  3. Memory organization, dream compression, or self-healing checkups are handled silently by low-cost models.

MCP Tool Assistant

bob does more than chat—he uses tools. Whether searching the web, scraping links, or running scripts, he automates the heavy lifting for you.

External Tools Auto Execution
  1. Follow the wizard in settings to connect your MCP standard plugin services.
  2. Ask bob, e.g.: "Search for the top 3 tech news from last night".
  3. bob silently invokes search tools, scrapes links, and presents the summary.

Long-term Memory

After every interaction, bob silently extracts facts and preferences in the background, consolidating them into long-term memory. Over time, he understands you better.

Long-term Memory Auto Organize
  1. Mention your project names, tech stack, or writing habits in normal chats.
  2. bob silently extracts facts and archives them into the local knowledge base.
  3. In future chats, bob automatically integrates long-term memory into his responses.

Local Knowledge Graph

Give him your files or quick notes, and bob weaves them into a clear, local knowledge graph. Use visual connections to trace back and retrieve your memories instantly.

Graph View Associative Search
  1. Drag a folder with Markdown notes into the chat box.
  2. Open the Graph View menu, bob renders the connections with a force-directed graph.
  3. Double-click any node to instantly jump to the related note and retrieve memories.

Goal-Oriented Execution

Enable Goal Mode to delegate vague objectives to bob. He breaks down steps, uses tools, and continuously verifies quality in the background until he delivers satisfactory results.

Task Breakdown Self-Verification
  1. Give bob a task objective and toggle on Goal Mode.
  2. The model decomposes the task (Maker) and invokes tools sequentially.
  3. The Clerk model (Checker) strictly verifies results, auto-retrying if failed.

Native Android, PC Connected

The ultimate mobile companion! Beyond seamless bidirectional SQLite synchronization over LAN or Relay tunnels, it now features powerful "PC Cross-Device Control". From anywhere, your phone can route commands directly to your computer via private encrypted tunnels. Your PC acts as a Headless Agent, silently executing complex tasks in the background and returning the results to your phone. True privacy, zero central cloud execution.

Headless PC Agent E2E Encrypted Tunnel Two-way Sync
  1. Summon the setup wizard on PC and scan the QR code to install the App.
  2. Scan the local network QR code to sync the SQLite DB and skills sandbox.
  3. Check notes and events offline; bidirectional merge happens automatically when online.

End-to-End Encrypted Web Drop

When transferring large files between phone and PC, bob establishes a secure peer-to-peer channel. Data flows only between your devices—never touching cloud servers. Ultimate security.

No Login Transfer Encrypted Privacy
  1. Click to share a large local file, bob auto-generates an AES key in memory.
  2. Scan the QR code with a phone or tablet on the same LAN to establish a WebRTC direct connection.
  3. Enjoy gigabit transfer speeds and instantly decrypt the file on the receiving end.

Self-Healing Doctor

bob has self-diagnostics and self-healing capabilities. If the network fluctuates, Keys expire, or configs conflict, he troubleshoots and attempts to fix it himself.

One-Click Checkup Self-Healing
  1. If the system feels abnormal, click the one-click Doctor checkup button in settings.
  2. The system automatically scans configs, database integrity, and API blockers.
  3. If data locks or conflicts occur, trigger self-healing to safely rollback from backups.

Smart Calendar & Reminders

Schedule time using natural language. bob organizes your weekly calendar locally and sends gentle reminders to your phone via WeChat when meetings or tasks approach.

Natural Language WeChat Alerts
  1. Casually say: "Check accounts with Lee from admin tomorrow at 3 PM".
  2. bob automatically extracts time and events and records them in the local calendar.
  3. As the schedule approaches, bob auto-dispatches a WeChat notification to your phone.

1. Deployment & Runtime: Zero Dependency & Invisible

Many Agents claiming to "run locally" are geek-driven tools requiring manual deployment of Node.js, Python, or CUDA. Some even force multi-GB downloads on first launch (e.g. Hermes Desktop downloading 2GB+ models). bob runs on Tauri v2 and Rust, statically compiling the core and local model sidecar into a single native binary of ~38MB, using only ~50MB memory. True zero-dependency installation and invisible presence.

2. Billing & Usage: Transparent & Autonomous, No Credit Traps

Enterprise workspaces often use closed Credit systems or expensive subscriptions. bob insists on the Bring Your Own Key (BYOK) open standard, charging zero markup. Freely configure raw API keys from DeepSeek, OpenAI, etc. to enjoy the lowest original Token pricing, or simply one-click import your own local models, completely eliminating quota anxiety.

3. Security & Control: No Overstepping, User Has Final Say

Visual Agents like Claude Computer Use constantly screenshot and hijack your physical mouse/keyboard, often executing erroneous commands due to slight UI shifts and causing file security panic. bob NEVER hijacks inputs. It restricts file writing through a visual Outbox sandbox and uses the system clipboard for rich text drafts. A human reviews and clicks "Send"—the perfect balance of efficiency and security.

4. Office-Specialized: Out-of-the-Box Skills for Professionals

Claude Code and Copilot are IDE plugins or terminal tools for programmers. bob comes with a practical toolchain specially tailored for general office professionals. Arrange schedules naturally, generate reminders, use WeChat remote control to send links/files to your PC, and enjoy Zero-Knowledge End-to-End encrypted large file sharing (Web Drop) on your local network. All daily routines wrapped up for you.

5. Agent Philosophy: Rejecting OS Hijacking and Blind Cloud Execution

Recent products like Tencent Marvis attempt to gain root-level permissions to directly control physical systems, which introduces immeasurable local security vulnerabilities despite the convenience. On the other extreme, Manus delegates all tasks (like scraping and coding) to an uncontrollable cloud sandbox, resulting in extremely high error-correction costs. bob strictly adheres to the "Human-in-the-Loop" boundary. It acts merely as your desktop's digital brain—never overstepping to click on your behalf, and never letting your sensitive data leave the local device. Human command, bob decision: the perfect unity of security and efficiency.
