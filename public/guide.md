# Bob Agent 使用指南

欢迎使用 Bob Agent，你的桌面 AI 助手。

---

## 快速开始

1. **配置模型**：在「设置 → 模型中心」中输入 API Key，Bob 即刻上线
2. **开始对话**：点击左侧「+ 新对话」按钮，输入你的问题
3. **切换对话**：左侧列表展示所有历史对话，点击即可切回

## 核心功能

### 多模型支持
Bob 支持多家模型供应商，你可以在「设置 → 模型中心」里自由切换：
- DeepSeek（推荐，性价比高）
- OpenAI（GPT-4o 系列）
- 通义千问、豆包、智谱、Kimi、MiniMax
- 也支持添加自定义 OpenAI 兼容接口

### 文件理解
直接把文件拖进聊天窗口，Bob 能帮你：
- 阅读和总结文档（TXT、PDF、Markdown）
- 分析图片内容（截图、照片）
- 处理代码文件

### 知识库（Wiki）
在「设置 → 知识库」中指定一个文件夹，Bob 会自动索引里面的文档，让回答更贴合你的业务场景。

### 工具能力
在「设置 → 凭证管理」中配置 API Key 后可解锁：
- **Web Search**：联网搜索最新信息（需 Tavily Key）
- **Web Fetch**：抓取网页正文内容（需 TinyFish Key）
- **External Skills**：加载外部认知技能包

### 全文搜索
在侧边栏的搜索框中输入关键词，Bob 会在所有历史对话中进行全文搜索，高亮匹配结果。点击搜索结果即可跳转到对应的对话。

### 记忆管理
在「设置 → 记忆管理」中查看和管理 Bob 积累的所有知识条目和对话记忆。你可以看到每条记忆的类型（知识库 / 会话记忆）和时间，也可以删除不需要的条目。

### 日程管理
在对话中说「帮我记一下明天下午三点开会」，Bob 会自动解析并创建日程提醒。

### Cron 定时任务
在「收件箱」中管理定时任务。当 Cron 任务执行完毕时，导航栏的收件箱图标会出现红点提示。

### 闪念速记
使用全局快捷键 `Ctrl+Shift+N` 快速唤起速记窗口，无需切换到主界面即可记录灵感。

### 晨报系统
每天打开 Bob 时，它会自动整理你近期的对话摘要，生成一份简洁的晨报。

## 外观定制

- **主题切换**：点击侧边栏顶部的太阳/月亮图标切换明暗模式
- **强调色**：在「设置 → 外观」中选择你喜欢的主题色
- **UI 缩放**：支持紧凑 / 标准 / 宽松三种信息密度

## 键盘快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl+Shift+B` | 从任何地方唤起 Bob 窗口 |
| `Ctrl+Shift+N` | 闪念速记（从任何地方快速记录） |
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行（不发送） |
| `F5` / `Ctrl+R` | 软重载（刷新对话列表和配置） |

## 数据存储

你的所有数据都保存在本机，不会上传到任何服务器：
- **配置文件**：`%APPDATA%\bob-agent\config.json`
- **对话数据库**：`%APPDATA%\bob-agent\bob.db`
- **知识库索引**：你指定的 Wiki 目录中

## 绿色版说明

如果你使用的是绿色版（解压即用版），Bob 的运行数据仍然存储在 `%APPDATA%\bob-agent\` 目录下。这意味着即使你把程序文件夹移动到其他位置，你的对话记录和配置都不会丢失。
