# bob-agent 产品需求文档

## 用户画像

| 维度 | 描述 |
|------|------|
| **是谁** | 办公室白领，对 AI 感兴趣但不愿折腾命令行 |
| **痛点** | ChatGPT 网页版不能读本地文件；API 工具太极客；国产大模型客户端体验参差 |
| **核心需求** | 开箱即用的对话 + 把本地文档/截图拖进去让 AI 分析/总结/改写 + 自动管理日程 |
| **红线** | 不能让用户碰命令行、不能让用户配 JSON、不能让用户懂 API |

#### 图片识别技术实现

DeepSeek V4 Pro/Flash 支持 Vision，通过 OpenAI 兼容 API 的 `image_url` 内容块传入 Base64 图片：

```javascript
messages.push({
  role: "user",
  content: [
    { type: "text", text: "帮我分析这张图片" },
    { type: "image_url", image_url: {
      url: `data:image/png;base64,${base64Data}`
    }}
  ]
});
```

图片来源与 Electron API 映射：

| 来源 | Electron API | 说明 |
|------|-------------|------|
| 剪贴板粘贴 | `clipboard.readImage()` | Ctrl+V 触发 |
| 文件拖拽 | HTML5 `dragenter/drop` + `FileReader` | 拖入窗口 |
| 全局截图 | `desktopCapturer` | Ctrl+Shift+S |
| 文件选择 | `dialog.showOpenDialog` | 点击 📎 按钮 |

#### 截图 → 日程管线（杀手功能）

```
用户截图微信群通知
  → Ctrl+V 粘贴
  → DeepSeek V4 Vision 识别图片文字
  → LLM Parser 提取: { type: "event", title: "产品评审会", start: "2026-05-15T14:00", location: "xx大厦" }
  → 弹出 ConfirmCard 组件
  → 用户点"确认"
  → 写入 SQLite + Microsoft 365 日历 + 设置提醒
```

#### Parser System Prompt（核心资产）

直接从 TodoList `parser.py` 移植，保持一致：

```
你是一个精准的日程解析助手。用户会发给你一段自然语言文字，你需要从中提取日程事件信息。

输出规则：
1. 必须输出严格的 JSON
2. 所有时间使用 ISO 8601 格式，时区默认 Asia/Shanghai
3. 未指定结束时间默认 +1 小时
4. 没有明确时间的待办任务 type 设为 "todo"

输出 Schema: { type, confidence, title, start_time, end_time, location, description, participants, priority, tags }
```

## 界面设计参考

```
┌─────────────────────────────────────────────┐
│  🤖 bob-agent            ─ □ ✕             │
├──────────┬──────────────────────────────────┤
│ 💬 对话   │  DeepSeek V4 Pro  ▾  💰 ¥0.12   │
│          │──────────────────────────────────│
│ 历史列表  │                                  │
│          │  👤 帮我分析一下这份销售数据        │
│ ○ 项目报告 │                                  │
│   分析    │  🤖 根据您上传的 Q3销售.xlsx:       │
│          │  ▸ 💭 思考过程 (点击展开)           │
│ ○ 合同审查 │                                  │
│          │  ## 核心发现                      │
│ ○ 周报撰写 │  1. 华东区增长 23%...              │
│          │  2. 毛利率环比下降 2.1%...          │
│          │                                  │
│──────────│──────────────────────────────────│
│ 📅 日程   │  📎 拖拽文件/图片   [发送]  ⚡模式  │
│ ⚙️ 设置   │                                  │
└──────────┴──────────────────────────────────┘
```

### 设计原则

1. **暗色优先**：默认暗色主题，降低视觉疲劳
2. **左右分栏**：左侧导航 + 右侧内容区
3. **圆角卡片**：所有内容块使用圆角卡片
4. **微动画**：消息出现、卡片展开使用 300ms ease 过渡
5. **零学习成本**：界面布局参考微信/飞书，用户无需学习
